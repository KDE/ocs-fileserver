## Dependencies

* apache2
* mysql-server
* php5
* php5-mysql
* php5-curl
* php5-gd
* ctorrent
* rtorrent
* transmission-daemon


## Installation

1. Set configuration options, if needed.

    api_application/configs/*

2. Deploy the project to web server

3. Change data directories to rewritable

    data/

    data/*
